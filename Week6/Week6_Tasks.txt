Задание 6. Ссылка,  динамические массивы, указатели на функцию, массив указателей на функцию.
1.	Реализовать 4 функции на простейшие арифметические операции (+,-,*,/). Каждая функция должна принимать два вещественных числа и возвращать результат операции в виде вещественного числа. Напишите функцию, которая принимает указатель на функцию в качестве параметра.
Добавьте меню, в котором пользователь может выбрать желаемую операцию. 

2.	Реализовать 3 функции каждая из которых принимает указатель на массив и количество элементов и возвращает указатель на новый массив. Первая функция ищет простые числа и заносит их в новый массив, вторая – все числа кратные трем, третья – все четные числа. Напишите функцию, которая принимает указатель на функцию в качестве параметра.
Добавьте меню, в котором пользователь может выбрать желаемую операцию. 
3.	Создать функцию, вставляющую строку в двумерный динамический массив в указанную позицию.
4.	Создать функцию, вставляющую столбец в двумерный динамический массив в указанную позицию.
5.	Создать функцию, удаляющую заданную строку двумерного динамического массива.
6.	Создать функцию, удаляющую заданный столбец двумерного динамического массива.
